# React shop
Frontend часть интернет-магазина


## Используемые технологии
1. React
2. Методология БЭМ
3. CSS Flexbox
4. CSS Grid

## Описание директории src
* `/components` - компоненты
* `/images` - папка с картинками
* `/utils` - папка с константами
* `/vendor` - папка со шрифтами и дополнительными библиотеками


## Установка и запуск проекта
* Клонировать репозиторий
* Установить зависимости `npm install`
* Запустить `npm run start`
* Выполнить сборку `npm run build`

## План доработки проекта:
* Доработать фильтр с категориями и ценой
* Реализовать добавление в корзину
* Добавить переход в корзину при клике на иконку
* Отрефакторить компоненты
